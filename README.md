# Minecraft Launcher Tweaker

A fun and powerful tool to customize and enhance your Minecraft launcher with cool features.

## Features

- Easy-to-use interface
- Customize launcher settings
- Performance optimization
- Theme customization
- Mod management support
- Advanced tweaking options

## Getting Started

### Prerequisites
- Java 11 or higher
- Python 3.8+

### Installation

```bash
git clone https://github.com/Aayush-Ranjan-26/minecraft-launcher-tweaker.git
cd minecraft-launcher-tweaker
pip install -r requirements.txt
```

### Usage

```bash
python launcher_tweaker.py
```
